# Jeopardy Bot

A cool slack jeopardy bot.

## Configuring

You can configure the bot via environment variables.
All of the configuration options (with one exception) are prefixed with `JBOT_` to prevent conflicts.

- JBOT_MODE
- JBOT_ROOMS
- JBOT_VERIFY_TOKENS
- JBOT_API
- JBOT_USERNAME
- MONGOLAB_URI
